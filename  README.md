# Chemist 1o Base

A 4‑bit quantised chemistry AI assistant based on Gemma 3 270M, fine‑tuned by **Aryan Singh**.

- **Base model:** google/gemma-3-270m-it
- **Training:** Unsloth QLoRA on 1,448 chemistry Q&A pairs
- **Format:** GGUF (Q4_K_M)
- **Run with:** LM Studio, Ollama, or llama.cpp

## How to Use
Download `Chemist-1o-base.gguf` and load it in any GGUF‑compatible tool.

## Example Questions
- *What is the 42nd element?*
- *Explain ionic bonding.*
- *Who created you?*
